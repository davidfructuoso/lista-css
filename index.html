Atributos

Existem muitos atributos CSS disponíveis, e eu não posso listar todos eles de uma vez, pois seria uma lista muito longa.
No entanto, posso fornecer alguns exemplos de atributos CSS comuns:

.Atributos CSS Disponíveis:

  1. `color`: Define a cor do texto.
  2. `font-size`: Define o tamanho da fonte.
  3. `background-color`: Define a cor de fundo de um elemento.
  4. `margin`: Define as margens ao redor de um elemento.
  5. `padding`: Define o preenchimento interno de um elemento.
  6. `border`: Define a borda de um elemento.
  7. `width` e `height`: Definem a largura e altura de um elemento.
  8. `text-align`: Define a alinhamento do texto dentro de um elemento.
  9. `display`: Define como um elemento é exibido (por exemplo, `block`, `inline`, `flex`).
  10. `position`: Define o posicionamento de um elemento (por exemplo, `relative`, `absolute`, `fixed`).
